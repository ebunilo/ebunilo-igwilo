# Image Placeholders and Specifications

Recommended Image Specifications:
Image Type	Dimensions	Format	Notes
Profile Photo	400x400px	JPG/PNG	Square, professional headshot
Architecture Diagrams	1200x800px	PNG	Use tools like draw.io, Excalidraw
Project Screenshots	1200x800px	PNG	Browser mockups work well
Favicon	32x32px	ICO/PNG	Simple icon or initials
